# Storybook project

## Getting started

## Technologies

- Node 17.3.0+
- NPM 8.3.0+
- Next.js 13.4.2
- Storybook 7.0.12

## Scripts

  ### `dev`
  
Starts the development server

  ### `storybook`

Starts the storybook server
  
  ### `build-storybook`

Build Storybook as a static web application

  ### `web-server`

Storybook will create a static web application capable of being served by any web server
